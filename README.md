# 🧠 Mentis Therapy AI

Uma plataforma SaaS inteligente voltada para psicanalistas, psicólogos, terapeutas, hipnoterapeutas e profissionais da saúde mental.  
O aplicativo utiliza Inteligência Artificial para auxiliar no atendimento clínico, transcrição de sessões, análise emocional, hipnose personalizada, relatórios automáticos e evolução terapêutica dos pacientes.

---

# ✨ Funcionalidades Principais

✅ Anamnese Psicanalítica Inteligente  
✅ Transcrição Automática de Sessões com IA  
✅ Relatórios Clínicos Automáticos  
✅ Assistente Terapêutico Inteligente  
✅ Hipnose Clínica Guiada por IA  
✅ Escalas Emocionais (Ansiedade, Depressão, Estresse etc.)  
✅ Plano Terapêutico Automático (8 a 15 sessões)  
✅ Evolução Emocional com Gráficos  
✅ Área Individual de Cada Paciente  
✅ Segurança LGPD e Criptografia  
✅ Dashboard Clínico Inteligente  
✅ Sistema SaaS Multiusuário
